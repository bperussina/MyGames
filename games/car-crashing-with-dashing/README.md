# car crashing with dashing

A standalone browser game — separate from Baby's Revenge 2 (its own port, link, and folder).

## Run

```bash
npm run dev -- car-crashing-with-dashing
```

## Play (built)

```bash
npm run play:car
# or
npm run play -- car-crashing-with-dashing
```

Opens on **port 5177** (Baby's Revenge 2 uses 5176).

## Family / iPad link

```bash
npm run family:car
```
