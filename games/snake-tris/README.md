# snake-tris

**Snake + CPU Tetris on the same screen.**

You play Snake — eat the dot, grow longer, score points. A computer plays Tetris on the **same grid**. Touch any Tetris block (falling or stacked) and you die.

## Controls

| Key | Action |
|-----|--------|
| Arrow keys / WASD | Move snake |
| R or Space | Restart |

## Run

```bash
npm run dev -- snake-tris
```

## Build

```bash
npm run build -- snake-tris
```
