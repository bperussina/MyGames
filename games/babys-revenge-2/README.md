# Baby's Revenge 2

The big kids are back! Survive **20 nights** using your **annoying sound maker** (hold **X**) to scare them away.

## Story

1. **Cutscene** — Baby dances with ducks 🐤
2. **Cutscene** — Big kids plot revenge and grab more toys
3. **Survive** — Day (3 min) and Night (3 min) cycles until Night 20

## Controls

| Input | Action |
|-------|--------|
| **Arrow keys** | Move around the nursery |
| **Hold left-click + drag** | Look around (like BR1) |
| **X** (hold) | Loud sound maker |
| **E** | Use toy box |
| **`** | Admin panel |

## Run

```bash
npm run dev -- babys-revenge-2
```

## Admin panel

Press **`** for god mode, skip nights, speed up time, and more.
