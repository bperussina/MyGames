# snake

Classic Snake — eat the food, grow longer, and do not hit the walls or yourself.

## Controls

- **Arrow keys** or **WASD** — move
- **R** or **Space** — restart after game over

## Run

```bash
npm run dev -- snake
```

## Build

```bash
npm run build -- snake
```

## Ideas to try

- Speed up as the snake grows.
- Add walls in the middle of the board.
- Copy this game and make a two-player version.
