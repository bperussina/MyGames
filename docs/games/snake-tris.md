# snake-tris

Snake and CPU Tetris on one screen. Eat dots, dodge Tetris blocks.

## Run

```bash
npm run dev -- snake-tris
```

## Rules

- You are the green snake
- Eat the pink dot to score
- Purple blocks are Tetris — played by the CPU
- Touch any Tetris block = game over
