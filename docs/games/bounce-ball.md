# bounce-ball

A simple bouncing ball game — the first example in this monorepo.

The ball bounces off the walls. Each bounce changes its color and adds to your score. Press **R** to reset.

## Run

```bash
npm run dev -- bounce-ball
```

## Build

```bash
npm run build -- bounce-ball
```

## Ideas to try

- Make the ball bigger or smaller when it bounces.
- Add a paddle you move with arrow keys.
- Spawn more balls!
- Import `random` from `@mygames/shared` for fun effects.
