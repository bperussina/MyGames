# snake

Classic Snake built with the mygames monorepo and `@mygames/shared` helpers.

Eat red food to grow and score points. Avoid walls and your own tail.

## Run

```bash
npm run dev -- snake
```

## Build

```bash
npm run build -- snake
```

## Controls

| Key | Action |
|-----|--------|
| Arrow keys / WASD | Move |
| R or Space | Restart after game over |
